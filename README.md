# 🚗 Parking Management System

A web-based Parking Management System built with HTML, CSS, and JavaScript. This project allows users to manage parking slots, book spaces, and track vehicle entries and exits efficiently.

---

## 📌 Features

- 🟢 Real-time parking slot availability (Available / Occupied / Reserved)
- 🚘 Vehicle booking with name, vehicle number, and entry time
- 💰 Automatic fee calculation based on duration
- 📋 Active bookings table with release option
- 🔒 Admin panel for managing slots
- 💾 Data persistence using localStorage
- 📱 Fully responsive design (Mobile + Desktop)

---

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling and animations
- **JavaScript** – Logic and interactivity
- **localStorage** – Data persistence

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/thousifsyed04/parking-management-system.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd parking-management-system
   ```

3. **Open the project:**
   - Simply double-click `index.html`
   - Or open it in any browser

> ✅ No installations required! It runs directly in the browser.
